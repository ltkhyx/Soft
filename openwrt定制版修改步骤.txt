在 OpenWrt 上让硬盘自动休眠，主要靠 hdparm 或 sdparm 来控制 SATA/USB 硬盘的省电模式。下面给你完整步骤
opkg update
opkg install hdparm
2️⃣ 查看硬盘信息
lsblk
# 或者
cat /proc/partitions

找出硬盘设备名，例如 /dev/sda1 设置自动休眠（spindown）
# 设置 10 分钟无操作自动休眠
hdparm -S 120 /dev/sda1
说明：
-S 120 → 单位为 5 秒 × 120 = 600 秒 = 10 分钟
/dev/sda → 硬盘设备

设置电源管理模式
# 启用高级电源管理（0-255）
hdparm -B 127 /dev/sda1
1-127 → 省电模式
128-254 → 性能模式
255 → 禁用自动休眠
hdparm -B 127 -S 120 /dev/sda1

永久生效
编辑 /etc/rc.local，在 exit 0 之前添加：
hdparm -B 127 -S 120 /dev/sda1

测试硬盘休眠
hdparm -C /dev/sda1
输出 standby 表示硬盘已进入休眠。
